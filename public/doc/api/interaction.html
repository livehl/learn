S.alert(msg, level, option)

警告框，返回值为Deferred对象

参数

参数名 类型 默认值 说明
msg String 显示的信息
level String info 提示的级别，可选值为info，warning，success，danger
option Object 
{
    title: '提示',
    btnName: '确定'
}
titlealert框的标题，btnNamealert框的确定按钮的显示文字。

示例

S.confirm(msg, option)

确认框，返回值为Deferred对象

参数名 类型 默认值 说明
msg String 提示的消息
option Object 
{
    title: "提示",
    sureBtnName: "确定",
    cancelBtnName: "取消",
    sign: "info"
}
sign的可选值为info，warning，success，danger

示例

S.notice(msg, level, holdTime)

信息通知

参数

参数名 类型 默认值 说明
msg String 提示的消息
level String info 提示的级别，可选值为info，warning，success，danger
holdTime int 3000 持续时间
