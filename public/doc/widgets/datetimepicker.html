描述

该控件扩展自bootstrap的datetimepicker控件，github地址为https://github.com/smalot/bootstrap-datetimepicker

定义

s='datetimepicker'

参数

参数配置可参考https://github.com/smalot/bootstrap-datetimepicker， 通过s-datetimepicker属性来配置参数。

默认参数名 类型 默认值 说明
format String yyyy-mm-dd 日期的格式
autoclose boolean true 当日期选择后是否关闭
language String zh-CN 语言
minView String month 日期时间选择器所能够提供的最精确的时间选择视图。
todayHighlight boolean true 是否高亮显示当前日期
